A chip-8 emulator built using JS

## Run Locally

Clone the project

```bash
  git clone https://github.com/theaniketnegi/chip8-js.git
```

Go to the project directory

```bash
  cd chip8-js
```

Start the server

```bash
  python -m server.py
```
